Aplicação mobile para a organização diária de atividades com React Native e Expo

funções:
1- Adiciona tarefas novas;
2- Cada tarefa recebe: nome e descrição;
3- Visualiza as tarefas em um campo logo abaixo;
4- Exclui tarefas quando finalizadas;
5- Conta quantas atividades foram finalizadas;
6- Frases diferentes da API Chuck Norris a cada 5s.

tecnologias:
1- React native Expo
2- React hook;
3- React Query;
4- TypeScript;
5- AsyncStorage;
6- Github.
